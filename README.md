# OpenMedical App

## Getting started

```shell
# optional create a git namespace dir and cd into it
mkdir -p $HOME/src/github.com/openmedicalearth
cd $HOME/src/github.com/openmedicalearth

# clone the repo
git clone git@github.com:openmedicalearth/openmedicalapp.git
cd openmedicalapp
```
